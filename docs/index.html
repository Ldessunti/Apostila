Estrutrura de páginas HTML

  • Tag HTML, e a tag que identifica o documento como um hipertexto html
  • Tag HEAD, é a tag que apresenta informações gerais sobre a página e de configuração
  • Tag TITLE, e a tag que informa o título de uma p[agina na barra do navegador
  • Tag BODY, e a tag que delimita o conteúdo da página

Tags de Texto

  • Tag Strike (Permite tachar algum trecho)
  • Tag Sub (Texto fica abaixo das demais partes do texto)
  • Tag Sup (Texto fica acima das demais partes do texto)
  • Tag Big (Modifica o tamnaho do texto deixando maior)
  • Tag Small (Modifica o tamnaho do texto deixando menor)
  • Tag TT (Mantem o mesmo espaçamento entre todos os caracteres)
  • Tag Pre
    (Exibe o texto como ele é digitado)
  • Tag Cite (Mantem o titulo de uma publicação em italico)
  • Tag Strong (Aplica negrito em uma frase)
  • Tag Em (Aplica italico em uma frase)
  • Tag Blockquote
    (Permite inclusão de uma citação longa ou de origem em outro site)

Funcionamento CSS

Existem 3 formas de colocar stilo CSS na sua página

  1. Associar o estilo ao elemento HTML;
  2. Seção de estilo com a Tag Style;
  3. Associar um arquivo de extensão .css" a página. Utilizando a tag link no head da página

Seletores CSS

Existem 3 tipos de seletores no CSS, são eles:

  1. Seletores de elemento:
  2. Seletor de id:
  3. Seletor de classe para aplicar estilo em elementos diversos, usa-se: .nome_da_classe;

Cores CSS

As cores podem ser definidas utilizando o formato RGB. Neste formato a intensidade da cor tem a intensidade definida entre 0 e 255. Onde 0 é 0% e 255 é 100%.

Exemplo de color: rgb(0,255,0). Cor verde em 100%
